Trader Behavior vs Market Sentiment Analysis
📌 Objective

Analyze how Fear/Greed sentiment affects trader performance and behavior.

Dataset
Bitcoin Market Sentiment (Fear/Greed)
Hyperliquid Trader Data

Methodology
Data cleaning and merging by date
Feature engineering (PnL, win rate, trade size)
Analysis based on sentiment (Fear vs Greed)
Trader segmentation (size, frequency, consistency)

Key Insights
Traders perform better during Greed periods
Trade size and frequency increase during Greed
Consistent traders show stable profitability

💡 Strategy Recommendations
Reduce risk during Fear periods
Increase trade size cautiously during Greed
